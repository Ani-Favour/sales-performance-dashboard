# Sales Performance Dashboard

## Overview
**This project presents an interactive sales performance dashboard analyzing
sales, orders, quantity, discounts, customer segments, and regional performance.**

## Key Insights
- **Total Sales:** $2.30M
- **Orders:** 5,009
- **Quantity Sold:** 37,873
- **Average Discount:** 15.62%
- **Top Performing State:** California (19.9%)
